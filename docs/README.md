# 자동차 경주 구현

1. 자동차의 이름을 한줄로 입력 받을 수 있다.
   - 자동차 이름은 콤마(,)로 구분할 수 있다.
   - 각 자동차의 이름은 5자 이하만 가능하다.
   - 자동차의 이름이 존재하지 않으면  
2. 시도할 횟수를 입력받을 수 있다.
3. 시도 횟수만큼 각 자동차들의 진행도를 출력한다.
   - 각 횟수마다 자동차들은 0~9사이의 랜덤 값에 따라 앞으로 전진하거나 그대로 멈춰있는다.
   - 랜덤 값이 4 이상이면 전진한다.
   - 랜덤 값이 3 이하이면 멈춘다. 
4. 최종 우승자를 찾아서 출력한다.
